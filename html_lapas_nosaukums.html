HTML Lapas nosaukums


Katrai tīmekļa lapai ir jābūt lapas nosaukumam, lai aprakstītu lapas nozīmi.



HTML Lapas nosaukums


Elements < title > pievieno lapai nosaukumu:


  Piemērs


< !DOCTYPE html >
< html >
< head >
< title >Mājaslapas nosaukums< /title >
< /head >
< body >

Teksts

< /body >
< /html>

Lapas nosaukums ir ļoti svarīgs meklētājprogrammu optimizācijai (SEO). Tekstu izmanto meklētājprogrammu algoritmi, lai noteiktu secību, uzskaitot lapas meklēšanas rezultātos.


Elements < title >:

    •  Definē nosaukumu pārlūkprogrammas rīkjoslā;
    •  Nodrošina lapas nosaukumu, kad tā tiek pievienota izlasei;
    •  Meklētājprogrammas rezultātos parāda lapas nosaukumu.
