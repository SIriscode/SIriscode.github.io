HTML Elementi un atribūti


HTML elementu nosaka sākuma tags, saturs un beigu tags.


Atribūti sniedz papildu informāciju par HTML elementiem, tie vienmēr ir norādīti sākuma tagā. Atribūti parasti ir nosaukuma/vērtības pāros, piemēram: name="value".



Saturs:

< a > atribūts href norāda tās lapas URL, uz kuru norāda saite.
< img > atribūts src norāda ceļu uz parādāmo attēlu.
< img > width un height atribūti nodrošina attēlu izmēru.
< img > alt atribūts nodrošina attēla alternatīvo tekstu.
style atribūts tiek izmantots, lai elementam pievienotu stilus, piemēram, krāsu, fontu, izmēru un vēl daudz ko citu.


Piemēram:


< tagname > Saturs būs šeit... < /tagname >


< h1 > Virsraksts < /h1 >


< p > Teksts < /p >


Piezīme:  Dažiem HTML elementiem nav satura (piemēram, elementam < br >(tas nosaka rindiņas pārtraukumu)). Šos elementus sauc par tukšiem elementiem. Tukšiem elementiem nav beigu taga!

Taču elementiem, kuriem ir saturs, vienmēr ir jābūt sākuma un beigu tagam.


Piemērs


< !DOCTYPE html >
< html >
< body >

< h1 >Mans pirmais virsraksts.< /h1 >
< p >Mans pirmais teksts.< /p >

< /body >
< /html >

Piemēra izskaidrojums:


    •  Elements < html > ir saknes elements, un tas nosaka visu HTML dokumentu. Tam ir sākuma tags < html > un beigu tags < /html >. Iekšā elementā < html > atrodas elements < body >.


    •  Elements < body > nosaka dokumenta pamattekstu. Tam ir sākuma tags < body > un beigu tags < /body >. Pēc tam elementā < body > ir vēl divi elementi: < h1 > un < p >.


    •  Elements < h1 > nosaka virsrakstu. Tam ir sākuma tags < h1 > un beigu tags < /h1 >.


    •  Elements < p > definē rindkopu. Tam ir sākuma tags < p > un beigu tags < /p >.



< !DOCTYPE > deklarācija


Deklarācija < !DOCTYPE > norāda dokumenta veidu un palīdz pārlūkprogrammām pareizi attēlot tīmekļa lapas.


Tam ir jāparādās tikai vienu reizi lapas augšdaļā pirms HTML tagiem.


Deklarācija < !DOCTYPE > nav reģistrjutīga.


< !DOCTYPE html >


HTML virsraksti


HTML virsraksti tiek definēti ar tagiem < h1 > līdz < h6 >.


< h1 > definē vissvarīgāko / vislielāko virsrakstu. < h6 > definē vismazāk svarīgāko virsrakstu:


Piemērs


< h1 >Virsraksts 1< /h1 >
< h2 >Virsraksts 2< /h2 >
< h3 >Virsraksts 3< /h3 >
< h4 >Virsraksts 4< /h4 >
< h5 >Virsraksts 5< /h5 >
< h6 >Virsraksts 6< /h6 >


HTML rindkopas


HTML rindkopas tiek definētas ar tagu < p >:


Piemērs


< p >Šī ir pirmā rindkopa< /p >
< p >Šī ir cita rindkopa< /p >


HTML saites


HTML saites tiek definētas ar tagu < a >:


Piemērs


< a href="saite.com" > Šī ir saite < /a >

Saites galamērķis ir norādīts atribūtā href.


Atribūti tiek izmantoti, lai sniegtu papildus informāciju par HTML elementiem.



HTML attēli


HTML attēli tiek definēti ar tagu < img >.


Avota fails (src), alternatīvais teksts (alt), width un height tiek nodrošināti kā atribūti:


Piemērs


< img src="img/zieds.jpg" alt="Zieds" width="250" height="160" >

Pieejami divi veidi, kā norādīt URL src atribūtā:


1. Absolūtais URL — saite uz ārējo attēlu, kurš atrodas citā vietnē.
Piemēram: src="https://www.vietne.com/images/dog.jpg".


Piezīmes. Ārējie attēli var būt aizsargāti ar autortiesībām. Ja nav saņemta atļauja to izmantot, iespējams, tiks pārkāptas autortiesības. Turklāt ārējos attēlus nevar kontrolēt, tos var pēkšņi noņemt vai mainīt.


2. Relatīvais URL — saites uz attēlu, kuras atrodas vietnē. Šeit URL neietver domēna nosaukumu.
Ja URL sākas bez slīpsvītras, tas būs relatīvs attiecībā pret pašreizējo lapu. Piemēram: src="dog.jpg".
Ja URL sākas ar slīpsvītru, tas būs saistīts ar domēnu. Piemēram: src="/images/dog.jpg".


Padoms. Gandrīz vienmēr vislabāk ir izmantot relatīvos URL. Mainot domēnu, tie nepazudīs.



Platuma un augstuma atribūti


Tagā < img > ir platuma un augstuma atribūti, kuri norāda attēla platumu un augstumu (pikseļos).




Style atribūts


Atribūts style tiek izmantots, lai elementam pievienotu stilus, piemēram, krāsu, fontu, izmēru un daudz ko citu.


Piemērs


< p style="color:green;"> Šis ir zaļš teikums. < /p >
