CSS background-attachment


CSS background-attachment rekvizīts kontrolē, vai fona attēls tiek ritināts vai paliek nemainīgs ar lapas saturu:



CSS background-attachment vērtības


  •     •  fixed - fona attēls paliek fiksēts, attēls ritināsies kopā ar pārējo saturu body sadaļā;

  •     •  local - ļauj fona attēlu ritināt kopā ar elementa saturu (attēls paliks tajā pat vietā, kur bija novietots);

  •     •  scroll - fona attēls paliks uz vietas (augšā), kad tas tiks ritināts uz leju (noklusējuma vērtība).


1. Piemērs


Šeit background-attachment saglabā fona attēlu fiksētu, ritinot pārējo saturu body sadaļā.


body {
    background-image: url("img/sievieteprogrammetaja.png");
    background-repeat: no-repeat;
    background-position: right top;
    background-attachment: fixed;
}

2. Piemērs


body {
    background-image: url("img/sievieteprogrammetaja.png");
    background-repeat: no-repeat;
    background-position: right top;
    background-attachment: local;
}

3. Piemērs


Šajā piemērā fona attēls paliks uz vietas, kad tas tiks ritināts uz leju (noklusējuma vērtība):


body {
    background-image: url("img/sievieteprogrammetaja.png");
    background-repeat: no-repeat;
    background-position: right top;
    background-attachment: scroll;
}
