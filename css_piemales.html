Piemales (margin)



Rekvizīts margin tiek izmantots, lai definētu atstarpi jeb telpu ap elementiem, ārpus noteiktām robežām. Tas ir pilnīgi caurspīdīgs un tam nav fona krāsas.



Rekvizīts margin ir šādu CSS rekvizītu saīsinājums:


    •  margin-top

    •  margin-bottom

    •  margin-right

    •  margin-left



margin rekvizītu var norādīt, izmantojot vienu, divas, trīs vai četras vērtības:


  •     •  Ja norāda vienu vērtību, tad tā piemēro vienu un to pašu piemali visām četrām pusēm;

  •     •  Ja norāda divas vērtības, tad pirmā vērtība attieksies uz augšējo un apakšējo daļu, bet otrā vērtība uz kreiso un labo daļu;

  •     •  Ja norāda trīs vērtības, tad pirmā vērtība attieksies uz augšējo daļu, otrā vērtība uz labo un kreiso daļu, trešā vērtība uz apakšējo daļu;

  •     •  Kad norāda visas četras vērtības, tad piemales attieksies uz augšējo, labo, apakšējo un kreiso daļu, pulksteņrādītāja secībā.


Piezīme:  Negatīvās vērtības tuvina elementu tā kaimiņiem, nekā tas būtu pēc noklusējuma vērtībām.


Visiem piemaļu rekvizītiem var būt šādas vērtības:


  •     •  auto - pārlūkprogramma aprēķinās rezervi;

  •     •  lenght - norāda malu px, pt, cm, utt.;

  •     •  % - norāda robežu % no saturošā elementa platuma;

  •     •  inherit - norāda, ka rezervei jābūt mantotai no vecākelementa.


1. Piemērs


div  {
   margin-top: 80px;
   margin-bottom: 50px;
   margin-right: 100px;
   margin-left: 70px;
}


Īsraksta rekvizīts - margin


2. Piemērs


Rekvizītam margin var norādīt vienu, divas, trīs vai četras vērtības:


p.one  {
   margin: 50px 70px 100px 90px;
}

p.two  {
   margin: 90px 50px 40px;
}

p.three  {
   margin: 50px 120px;
}

p.four  {
   margin: 40px;
}


Auto vērtība


Rekvizītu margin var iestatīt uz auto, lai elementu varētu centrēt horizontāli.


Vēlāk nepieciešams norādīt platumuPēc tam elements aizņems norādīto platumu, un atlikušā vieta tiks vienādi sadalīta starp kreiso un labo malu.
