HTML Krāsu vērtības


HTML krāsas var norādīt arī, izmantojot RGB, HEX, HSL, RGBA un HSLA vērtības.


Šīs vērtības būs tādas pašas kā krāsas nosaukums "Dodgerblue":


rgb(30, 144, 255)


#1e90ff


hsl(210, 100%, 56%)


Tie būs līdzīgi krāsas nosaukumam "Dodgerblue", tikai 50% caurspīdīgi:


rgba(30, 144, 255, 0.5)


hsla(210, 100%, 56%, 0.5)


Piemērs



< h2    style="background-color: rgb(30, 144, 255);" > ... < /h2 >
< h2    style="background-color: #1e90ff;" > ... < /h2 >
< h2    style="background-color: hsl(210, 100%, 56%);" > ... < /h2 >

< h2    style="background-color: rgba(30, 144, 255, 0.5);" > ... < /h2 >
< h2    style="background-color: hsla(210, 100%, 56%, 0.5);" > ... < /h2 >
