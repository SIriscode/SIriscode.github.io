HTML teksta formatēšana


Formatēšanas elementi tika izstrādāti, lai parādītu īpašus teksta veidus.




HTML teksta formatēšanas elementi:


< b > — treknraksts:

    •  Elements HTML < b > nosaka treknrakstu bez īpašas nozīmes.


< strong > — svarīgs teksts (treknraksts):

    •  HTML elements < strong > definē tekstu ar svarīgu nozīmi. Saturs tiek izcelts treknrakstā.


< i > — teksts slīprakstā

    •  HTML elementa < i > saturs parasti tiek rādīts slīprakstā. To bieži izmanto, lai norādītu tehnisku terminu, frāzi no citas valodas, domu, kuģa nosaukumu utt.


< em > — izcelts teksts

    •  HTML < em > elements definē uzsvērto tekstu. Saturs parasti tiek rādīts slīprakstā. Padoms: ekrāna lasītājs izrunās vārdus ar uzsvaru, izmantojot verbālo uzsvaru.


< mark > — atzīmētais teksts

    •  HTML elements < mark > nosaka tekstu, kas jāatzīmē vai jāizceļ.


< small > — mazāks teksts

    •  HTML elements < small > nosaka mazāku tekstu.


< del > — dzēsts teksts

    •  HTML elements < del > definē tekstu, kas ir izdzēsts no dokumenta. Pārlūkprogrammas parasti pārsvītro svītru caur izdzēsto tekstu.


< ins > — ievietots teksts

    •  HTML elements < ins > definē tekstu, kas ir ievietots dokumentā. Pārlūkprogrammas parasti pasvītro ievietoto tekstu.


< sub > — apakšraksta teksts

    •  HTML elements < sub > nosaka apakšindeksa tekstu. Apakšraksta teksts parādās pusi rakstzīmes zem parastās rindas, un dažreiz tas tiek atveidots mazākā fontā. Apakšraksta tekstu var izmantot ķīmiskām formulām, piemēram, H2O.


< sup > — virsraksta teksts

    •  HTML elements < sup > definē augšraksta tekstu. Virsraksta teksts parādās pusi rakstzīmes virs parastās rindas, un dažreiz tas tiek atveidots mazākā fontā. Virsraksta tekstu var izmantot matemātikā, piemēram, sin2.
