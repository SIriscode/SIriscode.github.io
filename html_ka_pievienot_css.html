Kā CSS var pievienot pie HTML



HTML dokuments tiks formatēts atbilstoši stila lapā esošajai informācijai, kad pārlūkprogramma nolasīs no tā informāciju.



Kas ir CSS?


Cascading Style Sheets (CSS) tiek izmantots, lai formatētu tīmekļa lapas izkārtojumu.

Izmantojot CSS, var kontrolēt krāsu, fontu, teksta lielumu, atstarpi starp elementiem, elementu izvietojumu un izkārtojumu, izmantojamos fona attēlus vai fona krāsas, dažādus displejus dažādām ierīcēm un ekrāna izmēriem un vēl daudz ko citu!



Lai sasaistītu CSS ar HTML var izmantot kādu no šiem veidiem:


    •  Inline - izmanto style atribūtu HTML elementā;

    •  Internal - izmantojot < style > elementu HTML faila sadaļā < head >;

    •  External - HTML dokumentu sasaistot, izmantojot elementu < link >, ar ārēju CSS failu.



Inline (Iekļautais stils) CSS


Iekļauto stilu var izmantot, lai vienam elementam piemērotu unikālu stilu.


Lai izmantotu iekļauto stilu, pievienojiet stila atribūtu attiecīgajam elementam. Stila atribūts var saturēt jebkuru CSS rekvizītu.


1. Piemērs


Iekļautie stili tiek definēti attiecīgā elementa atribūtā "style". Tas tiek izmantots, lai lietotu CSS vienā rindā vai elementā:


< h1  style="color:green; text-align:center;" > Šis ir virsraksts. < /h1 >
< p  style="color:red;" > Šis ir teksts. < /p >


Internal (Iekšējais stils) CSS


2. Piemērs


body {
    background-color: #89fbff;
}

h1 {
    background-color: #00807e;
}


External (Ārējais) CSS


Izmantojot ārējo stila lapu, var mainīt visas vietnes izskatu, mainot tikai vienu failu!


Ārējie stili ir definēti elementā < link >, HTML lapas sadaļā < head >:


3. Piemērs


Ārējo stila lapu var uzrakstīt jebkurā teksta redaktorā, un tas ir jāsaglabā ar .css paplašinājumu.


Ārējā .css failā nedrīkst būt HTML tagi.


< link  rel="stylesheet"  href="mystyle.css" >

Piezīme:  Nedrīkst pievienot atstarpi starp rekvizīta vērtību (20) un vienību (px):
Nepareizi (atstarpe): margin-left: 20 px;
Pareizi (bez atstarpes): margin-left: 20px;



Dažādas stila lapas


Ja vienam un tam pašam atlasītājam elementam ir definēti daži rekvizīti dažādās stilu lapās, tiks izmantota vērtība no pēdējās lasītās stila lapas.


Pieņemsim, ka ārējai stila lapai ir šāds < h1 > elementa stils:


h1 {
     color: navy;
}

Pēc tam pieņemsim, ka iekšējai stila lapai elementam < h1 > ir arī šāds stils:


h1 {
     color: orange;
}

Piemērs


Ja iekšējais stils ir definēts pēc saites uz ārējo stilu lapu, elementi < h1 > būs oranži:


< head >
< link  rel="stylesheet"  type="text/css"  href="mystyle.css" >
< style >
h1 {
      color: orange;
}
< /style >
< /head >

Piemērs


Tomēr, ja iekšējais stils ir definēts pirms saites uz ārējo stilu lapu, elementi < h1 > būs tumši zili:


< head >
 < style >
h1 {
      color: orange;
}
< /style >
< link  rel="stylesheet"  type="text/css"  href="mystyle.css" >
< /head >



Kaskadēšanas secība


Kāds stils tiks izmantots pirmais, ja HTML elementam ir norādīti vairāki stili?


Visi lapas stili tiks rādīti šādā secībā:


     1. Inline stils (HTML elementa iekšpusē);

     2. External un Internal stila lapas (head sadaļā);

     3. Pēc pārlūkprogrammas noklusējuma.


Tātad inline stilam ir visaugstākā prioritāte, un tas ignorēs external un internal stilus, un pārlūkprogrammas noklusējuma iestatījumus.
