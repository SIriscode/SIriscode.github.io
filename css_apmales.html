CSS apmales tekstam



CSS border rekvizīts ļauj norādīt elementa apmales stilu, platumu un krāsu.



Man ir apmales visās pusēs.


Man ir zila apakšējā apmale.


Man ir zaļa kreisā apmale un zaļš fons.


Man ir noapaļotas apmales oranžā krāsā no visām pusēm.




CSS apmaļu stils


border-style rekvizīts pievieno apmalēm stilu:


Apmalēm stilu var iestatīt ar šādām vērtībām:


  •     •  dotted - iestata punktētu apmali;

  •     •  solid - iestata vienkāršu līniju apmalei;

  •     •  dashed - definē pārtrauktu apmali;

  •     •  double - iestata dubulto apmali;

  •     •  ridge - definē 3D izciļņu apmali. (Efekts ir atkarīgs no apmales krāsas vērtības);

  •     •  groove - definē 3D rievotu apmali. (Efekts ir atkarīgs no apmales krāsas vērtības);

  •     •  outset - definē 3D sākuma apmali. (Efekts ir atkarīgs no apmales krāsas vērtības);

  •     •  inset - definē 3D ielaiduma apmali. (Efekts ir atkarīgs no apmales krāsas vērtības);

  •     •  hidden - definē slēpto apmali;

  •     •  none - nenosaka robezu apmales.


border-style rekvizītam var būt četras vērtības (augšējā apmale, labā apmale, apakšējā apmale, kreisā apmale).


Piemērs


Dažādu apmaļu stili:


p.dotted  { border-style: dotted; }
p.solid  { border-style: solid; }
p.dashed  { border-style: dashed; }
p.double  { border-style: double; }
p.ridge  { border-style: ridge; }
p.groove  { border-style: groove; }
p.outset  { border-style: outset; }
p.inset  { border-style: inset; }
p.hidden  { border-style: hidden; }
p.none  { border-style: none; }
