HTML Stils


HTML style atribūts tiek izmantots, lai elementam pievienotu stilu, piemēram, krāsu, fontu, izmēru un daudz ko citu.



HTML style atribūtam ir šāda sintakse:


< tagname style= "property:value;" >


Fona krāsa


CSS background-color rekvizīts nosaka HTML elementa fona krāsu.


Šeit tiks iestatīta lapas fona krāsa un arī tiks iestatīta fona krāsa diviem dažādiem elementiem.


Piemērs


< body style= "background-color: LightGreen;" >

< h1 style= "background-color: LightPink;" > Mans pirmais virsraksts.< /h1 >
< p style= "background-color: LightSkyBlue;" > Mans pirmais teksts.< /p >

< /body >


Teksta krāsa


CSS color rekvizīts nosaka HTML elementa teksta krāsu:


Piemērs


< h1 style= "background-color: LightPink;" > Mans pirmais virsraksts.< /h1 >
< p style= "background-color: LightSkyBlue;" > Mans pirmais teksts.< /p >


Fonti


CSS font-family rekvizīts nosaka fontu, kuru izmantos HTML elements.


Piemērs


< h1 style= "font-family:cursive;" > Mans pirmais virsraksts.< /h1 >
< p style= "font-family:fantasy;" > Mans pirmais teksts.< /p >


Teksta izmērs


CSS font-size rekvizīts nosaka HTML elementa teksta izmēru.


Piemērs


< h1 style= "font-size:250%;" > Šis ir virsraksts.< /h1 >
< p style= "font-size:120%;" > Šis ir teksts.< /p >


Teksta līdzinājums


CSS text-align rekvizīts nosaka HTML elementa horizontālo teksta līdzinājumu.


Piemērs


< h1 style= "text-align:center;" > Centrēts virsraksts.< /h1 >
< p style= "text-align:center;" > Centrēts teksts.< /p >
