Ievads HTML



HTML ("HyperText Markup Language") jeb Hiperteksta iezīmēšanas valoda. Tā ir standarta iezīmēšanas valoda tīmekļa lapu izveidei. Tā ļauj izveidot un strukturēt sadaļas, rindkopas un saites, izmantojot HTML elementus, piemēram, tagus un atribūtus.



HTML tiek uzskatīts par vienu no 3 galvenajiem rīkiem tīmekļa lapu izveidē:


    •  HTML nosaka struktūru, kā vietnē tiks attēlots teksts, attēli, tabulas un citi elementi.
    •  CSS nosaka šo elementu vizuālās īpašības, piemēram, izkārtojumu un krāsu.
    •  Javascript nodrošina dažādu elementu darbību / funkcionalitāti.



Sākotnējo HTML valodas versiju 1993. gadā uzrakstīja Tims Berners-Lī. Mūsdienās ir daudz dažādu HTML versiju.


2019.gada 28.maijā W3C paziņoja, ka WHATWG būs vienīgais HTML un DOM standartu izdevējs.


W3C un WHATWG ir publicējuši dažādus konkurējošus standartus kopš 2012.gada. Lai gan W3C standarts 2007.gadā bija identisks WHATWG, standarti kopš tā laika ir pakāpeniski sākuši atšķirties dažādu dizaina lēmumu dēļ.



HTML dokumentu sastāvdaļas ir elementi un tagi. Tie norāda pārlūkprogrammai, kā parādīt saturu. Tags sākas ar < html > un beidzas ar < /html > zīmēm. Burti starp zīmēm tiek saukti par elementu saturu.
