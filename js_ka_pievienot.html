Kā pie HTML var pievienot JavaScript?



Tags < script >


HTML valodā JavaScript kods tiek ievietots starp tagiem < script > un < /script >.


Piemērs


< script >
    document.getElementById("demo").innerHTML = "Mans pirmais JavaScript";
< /script >


JavaScript funkcijas un notikumi


JavaScript function ir JavaScript koda bloks, ko var izpildīt, kad tas tiek "izsaukts".


Piemēram, funkciju var izsaukt, kad notiek notikums, piemēram, kad lietotājs noklikšķina uz pogas.



JavaScript sadaļā < head > vai < body >


HTML dokumentā var ievietot neierobežotu skaitu skriptu.


Skriptus var ievietot HTML lapas sadaļā < body > vai < head >, vai abos.



JavaScript sadaļā < head >


Šajā piemērā JavaScript function ir ievietota HTML lapas sadaļā < head >.


Funkcija tiek izsaukta, kad tiek noklikšķināts uz pogas.


Piemērs


< head >
< script >
function myFunction() {
    document.getElementById("demo").innerHTML = "Teksts izmainījās";
}
< /script >
< /head >


JavaScript sadaļā < body >


Šajā piemērā JavaScript function ir ievietota HTML lapas sadaļā < body >.


Funkcija tiek izsaukta, kad tiek noklikšķināts uz pogas.


Skriptu ievietošana elementa < body > apakšā uzlabo attēlošanas ātrumu, jo skripta interpretācija palēnina displeju.


Piemērs


< body >
< script >
function myFunction() {
    document.getElementById("demo").innerHTML = "Teksts izmainījās";
}
< /script >
< /body >


Ārējais JavaScript


Ārējie skripti ir praktiski, ja viens un tas pats kods tiek izmantots vairākās tīmekļa lapās.


JavaScript failiem ir faila paplašinājums .js.


Lai izmantotu ārēju skriptu, skripta faila nosaukumu ievietojiet < script > taga atribūtā src (source).


Skripts darbosies tā, it kā tas atrastos tieši tur, kur atrodas tags < script >.


Piemērs


< script > src ="myScript.js" > < /script >


Ārējās JavaScript priekšrocības


    •  Tas atdala HTML un kodu;

    •  Tas atvieglo HTML un JavaScript lasīšanu un uzturēšanu;

    •  Kešatmiņā saglabātie JavaScript faili var paātrināt lapu ielādi;


Ārējās atsauces


Uz ārējo skriptu var atsaukties 3 dažādos veidos:


    •  Ar pilnu URL (pilnu tīmekļa adresi);

    •  Ar faila ceļu (piemēram, /js/);

    •  Bez jebkāda ceļa;
