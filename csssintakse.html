CSS sintakse



CSS sintakse attiecas uz veidu, kā mēs rakstām CSS kodu. Lai rakstītu CSS, vispirms savā HTML lapā ir jāidentificē elements, kuram vēlamies izveidot stilu.



CSS sastāv no stila noteikumiem, kurus piemēro norādītajiem dokumenta elementiem un vēlāk tos interpretē pārlūkprogrammā.




Stila noteikums sastāv no 3 daļām:


  •     •  Atlasītājs (selector) - norāda HTML elementu, kuram vēlamies pievienot stilu. Tas var būt jebkurš elements, piemēram, < h2 > vai < div >;

  •     •  Rekvizīts (property) - tas ir iebūvēts CSS rekvizīts, ko izmanto elementu stila veidošanai;

  •     •  Vērtība (value) - tas ir ko vēlamies pievienot CSS rekvizītam. Piemēram, krāsu rekvizīta vērtība var būt gan red, gan #ff0000;

  •     •  Deklarācija (declaration) - sastāv no CSS rekvizīta un CSS vērtības, kuri ir atdalīti ar kolu viens no otra(:). Vairākas CSS deklarācijas ir atdalītas ar semikolu (;). Visi deklarācijas bloki ir ietverti cirtainajās iekavās({}).


Piemērs


Visi < p > elementi būs zilā krāsā un teksts būs centrēts:


p {
    color: blue;
    text-align: center;
}

Piemēra izskaidrojums:


  •     •  p ir CSS atlasītājs, kurš norāda uz HTML elementu, kura stils tiks mainīts;

  •     •  color ir rekvizīts, un blue ir rekvizīta vērtība;

  •     •  text-align ir rekvizīts, un center ir rekvizīta vērtība;

  •     •  font-size ir rekvizīts, un 21px ir rekvizīta vērtība.
