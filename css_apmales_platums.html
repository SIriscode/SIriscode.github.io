CSS apmaļu platums



Rekvizīts border-width iestata elementa apmales platumu.



Rekvizīts border-width ir šādu CSS rekvizītu saīsinājums:


  •     •  border-bottom-width;

  •     •  border-left-width;

  •     •  border-right-width;

  •     •  border-top-width;




border-width rekvizītu var norādīt, izmantojot vienu, divas, trīs vai četras vērtības:


  •     •  Ja norāda vienu vērtību, tā piemēros vienādu platumu visām četrām malām;

  •     •  Ja norāda divas vērtības, tad pirmais platums attieksies uz augšējo un apakšējo malu, otrais platums tiks iestatīts kreiso un labo malu;

  •     •  Ja norāda trīs vērtības, pirmais platums attieksies uz augšējo malu, otrais platums uz kreiso un labo pusi, trešais platums uz apakšējo malu;

  •     •  Kad norāda visas četras vērtības, platumi attiecas uz augšējo, labo, apakšējo un kreiso malu (pulksteņrādītāja virzienā).



Platumu var iestatīt kā konkrētu izmēru, piemēram, pikseļos, pt, cm, em vai izmantojot vienu no trim iepriekš definētajām vērtībām: thin (tievs), medium (vidējs) vai thick (biezs):



Piemērs


p.one  {
border-style: solid;
border-width: 2px;
}

p.two  {
border-style: solid;
border-width: 0.5em;
}

p.three  {
border-style: solid;
border-width: 4px 1.25em;
}

p.four  {
border-style: solid;
border-width: 3ex 1.25ex 1ex;
}

p.five  {
border-style: solid;
border-width: 0 4px 8px 12px;
}

p.six  {
border-style: solid;
border-width: medium;
}
