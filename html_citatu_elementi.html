HTML Citātu elementi



< blockquote >:

    •  HTML elements < blockquote > nosaka sadaļu, kas tiek citēta no cita avota.


< q >:

    •  HTML tags < q > nosaka īsu citātu. Pārlūkprogrammās parasti ievieto pēdiņas ap citātu.


< abbr >:

    •  HTML tags < abbr > definē saīsinājumu vai akronīmu, piemēram, "HTML", "CSS". Marķējuma saīsinājumi var sniegt noderīgu informāciju pārlūkprogrammām, tulkošanas sistēmām un meklētājprogrammām. Padoms: Izmantojiet globālā nosaukuma atribūtu, lai parādītu saīsinājuma/akronīma aprakstu, kad virzāt peles kursoru virs elementa.


< address >:

    •  HTML tags < address > nosaka dokumenta vai raksta autora/īpašnieka kontaktinformāciju. Kontaktinformācija var būt e-pasta adrese, URL, fiziskā adrese, tālruņa numurs utt. Teksts elementā < address >> parasti tiek atveidots slīprakstā, un pārlūkprogrammas vienmēr pievieno rindiņas pārtraukumu pirms un pēc šī elementa.


< cite >:

    •  HTML tags < cite > nosaka radošā darba nosaukumu (piemēram, grāmatas, dzejoļa, dziesmas, filmas, gleznas, skulptūras utt.). Teksts parasti tiek atveidots slīprakstā.


< bdo >:

    •  BDO apzīmē Bi-Directional Override. HTML tags < bdo > tiek izmantots, lai ignorētu pašreizējo teksta virzienu.
